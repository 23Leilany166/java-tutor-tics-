Bienvenida

¿Qué vas a hacer en este sitio?

Aquí encontrarás una colección de ejemplos y problemas de programación estructurada en Java. La idea es que cada estudiante elija un ejercicio, analice el enunciado, escriba su solución en código y luego la valide con pruebas automáticas (tests).

¿Cómo trabajar con los ejercicios?

  • 1. Elige un ejercicio desde el menú de arriba o desde la lista de abajo.
  • 2. Lee el enunciado, el pseudocódigo, el diagrama de flujo y el código ejemplo.
  • 3. Programa tu solución en Java en tu proyecto de Maven/IDE.
  • 4. Corre los tests (JUnit) para comprobar si tu programa funciona correctamente.
  • 5. Si un test falla, revisa el algoritmo, corrige el código y vuelve a ejecutar las pruebas.

El objetivo no es solo que el programa “corra”, sino que aprendas a pensar en pasos, usar variables, ciclos, decisiones y estructuras de datos como vectores y matrices, y comprobar tu lógica con pruebas automáticas.

Consejos rápidos

  • Empieza por los ejemplos (Ejemplo 1.7, Ejemplo 3.5) para calentar.
  • Continúa con problemas cortos (1.8, 2.3, PS 2.8–2.11).
  • Luego intenta ejercicios con arreglos y matrices (3.16, 4.3, 4.16, 4.21, 4.37, PS 4.5).
  • Al final, prueba los de estadística con listas de datos (PS 3.27, PS 3.28, PS 3.29).
